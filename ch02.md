# var, let, const

- let, const: 块级作用域
- var: 全局作用域

```js
{
  let a = 10;
  var b = 1;
}
console.log(a); // ReferenceError: a is not defined
console.log(b); // 1
```

```js
var a = [];
for (var i = 0; i < 10; i++) {
  a[i] = function () {
    console.log(i); // 全局的同一个 i
  };
}
a[6](); // 10
```

```js
var a = [];
for (let i = 0; i < 10; i++) {
  a[i] = function () {
    console.log(i); // 独立的 i
  };
}
a[6](); // 6
```

- var 有变量提升: 允许变量在初始化前访问
- let, const 没有变量提升

```js
console.log(foo); // undefined
var foo = 2;

console.log(bar); // ReferenceError: bar is not defined
let bar = 2;
```

### 暂时性死区

如果块中有 let 或 const 关键字, 则不允许变量在初始化前访问 (包括 typeof)

```js
var tmp = 123;

{
  // 暂时性死区 Temporal Dead Zone, TDZ
  tmp = "abc"; // ReferenceError: Cannot access 'tmp' before initialization
  let tmp;
}
```

```js
typeof x; // ReferenceError: x is not defined
let x;
```

```js
var y = 777;
{
  // 暂时性死区 Temporal Dead Zone, TDZ
  typeof y;
  let y; // ReferenceError: Cannot access 'y' before initialization
}
```

如果一个变量没有初始化, 反而不会报错

```js
typeof wtf; // undefined
```

```js
// 参数默认值的赋值顺序: 从左到右 x = y; y = 2;
// ReferenceError: Cannot access 'y' before initialization
function bar(x = y, y = 2) {
  return [x, y];
}
bar();
```

```js
var x = x; // 不报错
let x = x; // 报错 ReferenceError: x is not defined
```

let 不允许在同一个作用域中, 重复创建同一个变量

```js
{
  let a = 10;
  var a = 1; // SyntaxError: Identifier 'a' has already been declared
}

{
  let a = 10;
  let a = 1; // SyntaxError: Identifier 'a' has already been declared
}
```

let 不允许在函数体中重复创建参数

```js
function func(arg) {
  let arg; // Identifier 'arg' has already been declared
}
func(); // 报错

function func(arg) {
  {
    let arg;
  }
}
func(); // 不报错
```

### 为什么需要块级作用域

场景 1: 内层 var 变量可能覆盖外层 var 变量

```js
var tmp = new Date();

function f() {
  // var 变量提升, 输出 undefined
  console.log(tmp);
  if (false) {
    //
    var tmp = "hello";
  }
}

f(); // undefined
```

场景 2: i 泄露为全局变量

```js
var s = "hello";

for (var i = 0; i < s.length; i++) {
  console.log(s[i]);
}

console.log(i); // 5
```

### ES6 的块级作用域

内层作用域可以重新创建外层作用域的同名变量

```js
let name = "hello";
{
  let name = "world";
}
console.log(name);
```
