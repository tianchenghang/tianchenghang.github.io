### Babel 转码器 (ES6 => ES5)

```js
// 转码前
input.map((item) => item + 1);

// 转码后
input.map(function (item) {
  return item + 1;
});
```

```shell
# 安装 Babel
pnpm install -g @babel/core
# 安装转码规则
pnpm install -g @babel/preset-env
# 安装命令行转码工具
pnpm install -g @babel/cli

pnpm add --save-dev @babel/core @babel/preset-env @babel/cli
```

配置文件 .babelrc

```json
{
  "presets": ["@babel/preset-env"],
  "plugins": []
}
```

使用命令行转码工具

```shell
# 转码结果输出到 stdout
npx babel example.js
# 转码结果输出到文件
# --out-file 或 -o 指定输出文件
npx babel example.js --out-file compiled.js
npx babel example.js -o compiled.js
# 转码整个目录
# --out-dir 或 -d 指定输出目录
npx babel src --out-dir lib
npx babel src -d lib
# -s 生成 source map 文件
npx babel src -d lib -s
```

### Polyfill

Babel 默认只转码新语法 (Syntax), 不转码新 API

例如, ES6 为 Array 对象新增了 Array.from 方法, 而 Babel 不会转码该方法

解决: 使用 Polyfill 垫片, 为旧浏览器提供新的原生功能
