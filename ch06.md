# 正则表达式

```js
let regex = /xyz/i;
console.log(regex);

let regex1 = new RegExp("xyz" /* 字符串 */, "i" /* 正则表达式的修饰符 flag */);
console.log(regex1);

// 返回传递的正则表达式的拷贝
let regex2 = new RegExp(/xyz/i /* 正则表达式 */);
console.log(regex2);

// 替换传递的正则表达式的修饰符 flag
let regex3 = new RegExp(/xyz/i, "ig");
console.log(regex3, regex3.flags); // /xyz/gi gi
```
