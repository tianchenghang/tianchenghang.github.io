# 运算符

### 指数运算符 \*\*

```js
2 ** 3; // 8
2 ** (3 ** 2);
// 等价于
2 ** (3 ** 2);
let a = 4;
a **= 3;
// 等价于
a = a * a * a;
```

### 链式判断运算符 ?.

- obj?.prop 对象属性是否存在
- obj?.[expr] 对象属性是否存在
- func?.(...args) 函数或方法是否存在

```js
obj?.prop;
// 等价于
obj == null ? undefined : obj.prop;

obj?.[expr];
// 等价于
obj == null ? undefined : obj[expr];

obj?.meth();
// 等价于
obj == null ? undefined : obj.meth();

func?.();
// 等价于
func == null ? undefined : func();
```

```js
let data = {
  "user data": {
    ["first-name"]: "hang",
  },
};
let firstName = data?.["user data"]?.["first-name"] || "username";
console.log(firstName); // hang

data = {
  meth: {
    ["print-meth"](...args) {
      console.log(...args);
    },
  },
};
// 如果 ? 左侧为 null 或 undefined, 则直接返回 undefined
firstName = data?.["user data"]?.["first-name"] ?? "username";
console.log(firstName); // username

// 如果 ? 左侧方法存在, 则执行
data?.meth?.["print-meth"]?.("w", "t", "f"); // w t f

data = {};
data?.meth?.["print-meth"]?.("w", "t", "f") || console.log("404 not found"); // 404 not found
```

### null 判断运算符 ??

```js
x ||= y;
// 等价于
x || (x = y);

x &&= y;
// 等价于
x && (x = y);

x ??= y;
// 等价于
x ?? (x = y);
```
